## Basic Policy
This project aims to keep additions and changes as simple and minimal as possible.

## Communication
Please use Japanese exclusively when communicating with me.

## Development Commands
- `npm run build` - TypeScriptをコンパイル
- `npm run dev` - 開発モードで実行

## Tech Stack
- ts-morph: TypeScript AST解析
- better-sqlite3: データ永続化
- commander: CLI構築

## Commit Convention
Conventional Commits形式を使用（feat:, fix:, chore: など）

## Structure
- src/parser/ - AST解析ロジック
- src/cli.ts - CLIエントリーポイント
- src/types.ts - 型定義
